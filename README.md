# Surveys API
**Environment**

- Node: v16.13.0
- npm: 8.1.4

**don't forget to run 😁**

	npm install

**On Windows**:

**Run in production mode**

    npm start

**Run in dev mode**

    npm run dev_win

**Run tests**

    npm run test_win

**On Linux & Mac**:

**Run in production mode**

    npm start

**Run in dev mode**

    npm run dev

**Run tests**

    npm test
